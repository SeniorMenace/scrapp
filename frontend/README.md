# Multi Market Price Scraper Frontend

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```

## Environment Variables
- `NEXT_PUBLIC_API_URL`: Backend API endpoint (default: http://localhost:4000/api/search)

## Features
- Search input, search button, loading indicator
- Results grid with product cards
- Price sorting, marketplace filtering, in-stock filter
- Responsive design
- Error handling and empty state
